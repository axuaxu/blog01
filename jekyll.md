## Jekyll ##
 
### Ruby RubyDevKit Gem ###


### [Quick-start guide](http://jekyllrb.com/docs/quickstart/) ###

-  $ gem install jekyll
-  $ jekyll new myblog
- $ cd myblog
- /myblog $ jekyll serve
- => Now browse to http://localhost:4000
